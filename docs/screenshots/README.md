# Screenshots

Generate fresh portfolio screenshots after starting the web preview:

```bash
npm run web:export
npm run web:serve
```

Open `http://127.0.0.1:8081` in a browser and capture:

- Feed with AI scenario overlay
- Symbol page for `/symbol/NVDA`
- Post detail page for `/post/p1`
- Create thesis preview

Use these images in the GitHub README or CV project write-up after the first Vercel deployment URL is live.
